## Commit Message
Imperative verb (stem verb in active tone) + object

Good:
- `Add CONTRIBUTING.md`

Bad:
- `Added CONTRIBUTING.md`
- `CONTRIBUTING.md`
- `some file`
- `new file`
- `contributing instruction`
